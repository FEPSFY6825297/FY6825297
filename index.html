Abstract

            Digital data production is growing at unprecedented rates, meaning current methods of data storage are becoming
            Insufficient to keep up with the amount of data that needs to be stored. While still a developing field,
            a promising solution to this issue is DNA storage. DNA storage achieves a much higher storage density (Buko, T., Tuczko, N. and Ishikawa, T. 2023)
            and has promising potential in terms of durability. Converting digital data into DNA is a process based on binary code
             where a specific nucleotide relates to a specific code (00=Adenine,01=cytosine...). The digital data is encoded and translated into a DNA sequence
             using computer algorithms and then synthesised. In vivo methods of storage see DNA data storage coming closer to realising its potential for Long-term storage. 
             To extract digital information from DNA storage, The DNA needs to be sequenced (a process used to identify the sequence of bases in a DNA sequence) and then decoded
             to return the digital data back to its original form.
